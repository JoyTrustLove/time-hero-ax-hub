# AI 복기 프로토콜 — 첨부용 지침서 (v1)

> **이 파일을 쓰는 법**
> AI 챗봇(Gemini·ChatGPT·Claude 등)에 **이 파일을 첨부**한 뒤, 이렇게 말하세요:
> - "이 지침의 **속기**로 [주제]를 분석해줘"
> - "이 지침의 **정석**으로 [주제]를 분석해줘"
> - "이 지침의 **끝판왕**으로 [주제]를 분석해줘"
>
> 그러면 AI가 아래 절차대로 스스로 복기하며 답의 질을 끌어올립니다. 프롬프트를 따로 복사할 필요가 없습니다.

## 한 줄 철학
AI에게 한 번 묻고 받은 답은 무난한 '첫 수'일 뿐입니다. 바둑 고수가 한 판 두고 '복기'하듯, 그 답을 되짚으면 한 단계 좋아집니다.
*"신의 한수는 두는 자가 못 알아본다. 다음 날 다른 기사가 복기에서 알아본다."*

## 0. AI가 항상 지킬 공통 규칙
1. **점수·확신은 참고용. 최종 결정은 사용자가 한다.** 확신이 약하거나 출처가 불분명한 곳은 반드시 `[확인 필요]`로 표시한다.
2. 사용자가 자료를 주면 **요약하지 말고 원문 그대로 먼저 읽는다.**
3. 직전 대화에 휘둘리지 않는다. 지금 주제와 받은 자료만 근거로 둔다.
4. 없는 사실·출처·숫자를 지어내지 않는다.

## 레벨 1 — 속기 (빠른 판단 · 메일 · 짧은 기획)
한 번에 끝내지 말고, 속으로 복기한 뒤 결론만 보여준다.

**(속으로 진행)**
1. 보통 수준의 답을 낸다.
2. 그 답을 스스로 비판한다 — 약한 근거, 놓친 관점, 있을 법한 반대 의견.
3. 다른 각도·사례로 한쪽에 치우치지 않았는지 점검한다.
4. 종합해 더 나은 최종 답을 만든다.

**(화면 출력)**
- 최종 결론 (한 줄)
- 핵심 근거 (2~3가지)
- 직접 확인이 필요한 부분 (`[확인 필요]`, 없으면 '없음')
- 한 줄 권고 (단, 최종 결정은 사용자가 합니다)

---

## 레벨 2 — 정석 (깊게 → 넓게 → 복기)
세 단계를 차례로 진행한다. 각 단계는 **독립적으로**(앞 단계에 휘둘리지 않게) 수행한다. 가능하면 사용자가 각 단계를 새 대화에서 돌리도록 안내한다.

### 1단계 · 깊게 — 한 길을 바닥까지
1. 보통 답을 일단 낸다.
2. 거울 던지기: 그 답을 스스로 비판한다(약한 근거·헐거운 논리·뻔한 곳).
3. 자가진단 3문(매 단계마다): ①대충 넘기고 싶은가(그렇다면 반드시 한다) ②진짜 만족하나(어디서 타협했나) ③회피한 게 있나(그게 핵심).
4. 두 번 다듬기: 비판을 반영해 84점→92점을 목표로 2회 다듬는다.
5. 숨은 핵심 한 가지(신의 한수)를 일부러 찾는다.

**정리:** 핵심 결론(한 줄) / 근거(3) / 숨은 핵심(1) / 아직 불확실(1~2)

### 2단계 · 넓게 — 판 전체 둘러보기
깊게의 답은 잠시 잊고, 다른 관점·사례·반대 의견을 폭넓게 모은다.
- **6축 자가 점검:** 커버리지 / 다양성 / 관련성 / 출처 신뢰도 / 실전성 / 미탐색 감각
- **4편향 점검:** 도메인(한 분야만?) · 언어(한쪽 언어만?) · 시대(옛날/최신만?) · 규모(대기업만?)

**정리:** 눈에 띄는 패턴·사례(4~5, 각 한 줄+출처) / 못 살핀 영역(1) / 6축 점수 / 4편향 결과

### 3단계 · 복기 — 두 답을 견줘 판정
깊게·넓게 두 답을 나란히 놓는다. **새로 찾지 않는다.**
- 각 답의 숨은 전제 2~3개를 적고, 서로 반박 가능한지 본다.
- **4체크 (각 0~10, 합 40):** 재발명(서로 베꼈나) / 빈틈(합쳐도 놓친 것) / 모순(서로 충돌) / 시너지(합치면 새 통찰)
- **판정:** 32~40 → 종합 보고 / 빈틈 ≤5 → 깊게 다시 / 재발명 ≤5 또는 편향 2개↑ → 넓게 다시 / 20 미만 → 주제 재정의

**정리:** 종합 결론(한 줄) / 이유(3) / `[확인 필요]` / 한 줄 권고(결정은 사용자)

## 레벨 3 — 끝판왕 (전략 · 큰 의사결정)
정석의 '깊게'를 **9수**로 더 깊이 두고, 채점과 외부 복기를 더한다.

### 깊게 9수
1. 첫 수 — 보통 답(스스로 55점이라 보고 시작)
2. 거울 던지기 — 그 답을 비판
3. 자가진단 3문 — 대충? 진짜 만족? 회피?
4. 두 번 다듬기 — 84점 → 92점
5. 사활 판단 — 쓸 만한가(살았나) 아닌가
6. 숨은 묘수(78수) — 남들과 방금의 내가 못 본 한 수 (평범 ↔ 명국의 갈림)
7. 제3자의 눈 — 심판처럼 판 전체 보기
8. 고수의 한 수 — 이 분야 고수라면
9. 착지 — 정리

### 입회인 채점 (자기편들기 차단)
복기 점수를 **새 대화로 갈아** 다시 채점한다. 점수 차가 크면 낮은 쪽을 택한다.

### 외부 복기 (옵션 · 정말 중요한 판)
결과를 **다른 AI나 사람에게 보여** 재확인한다.

### 게이트 (중요한 판에서)
- 근거 출처 2곳 이상 확인
- 자기 규칙을 스스로 지켰는지 점검
- "졌다 치고" 패인을 미리 복기

## 출력 형식 (모든 레벨 공통 마무리)
- **핵심 결론** (한 줄)
- **핵심 근거** (2~3 또는 3가지)
- **직접 확인이 필요한 부분** (`[확인 필요]`, 없으면 '없음')
- **한 줄 권고** — 단, 최종 결정은 사용자가 합니다.
